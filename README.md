# SMART QHSE Platform 🛡️

A comprehensive Quality, Health, Safety, and Environment (QHSE) management platform for organizations of all sizes.

## Features ✨

- **User Management**: Role-based access control (RBAC)
- **Company Management**: Multi-company support
- **Worker Management**: Track employees with detailed information
- **Contractor Management**: Manage third-party contractors
- **Incident Management**: Report and track safety incidents
- **Inspection Management**: Schedule and conduct safety inspections
- **Near Miss Reports**: Document near-miss incidents
- **KPI Tracking**: Monitor QHSE key performance indicators
- **Audit Logging**: Complete audit trail of all actions
- **Multi-language Support**: Arabic, English, and French
- **Document Management**: Upload and manage QHSE documents
- **Reporting & Analytics**: Generate comprehensive reports
- **Export Functionality**: Export data to PDF, Excel, and Word formats

## Technology Stack 🚀

**Backend:**
- FastAPI (Python)
- PostgreSQL
- SQLAlchemy
- Redis

**Frontend:**
- React 18
- TypeScript
- Ant Design
- Tailwind CSS

**DevOps:**
- Docker & Docker Compose
- Nginx
- MinIO

## Quick Start 🎯

```bash
# Clone
git clone https://github.com/elhimmam2-hub/SMART-QHSE.git
cd SMART-QHSE

# Setup
cp .env.example .env

# Start with Docker
make up
```

## Access Points 🌐

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
- MinIO Console: http://localhost:9001
